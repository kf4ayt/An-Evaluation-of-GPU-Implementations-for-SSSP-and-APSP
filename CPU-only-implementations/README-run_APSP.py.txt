
Name: run_APSP.py

Purpose: This script runs the Floyd-Warshall APSP algorithm from the
         SciPy package on the specified graphs.

Requirements: This Python script requires Python 3 and the following
              packages: time, numpy, scipy.

Usage: Once Matrix Market graphs have been generated by the
       'generate_graph.py' utility, one need only edit the list that
       contains the number of vertices so as to specify which graphs
       to process. There are further notes in the script, itself,
       which should provide the remaining details for usage.
